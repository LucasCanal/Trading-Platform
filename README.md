# 📈 TradePro - Professional Trading Platform

TradePro é uma plataforma moderna de negociação de ativos digitais e inteligência de mercado. O projeto foi desenvolvido com foco em alta performance visual (UI/UX), oferecendo uma experiência fluida para traders profissionais e iniciantes.

## ✨ Características

- **Dashboard Avançado**: Visualização de estatísticas de mercado em tempo real com gráficos interativos.
- **Interface Responsiva**: Design totalmente adaptado para Desktop e Mobile, garantindo acessibilidade em qualquer dispositivo.
- **Sistema de Autenticação**: Fluxo completo de Login e Criação de Conta integrado ao backend.
- **Gestão de Carteira**: Quick Transfer para contatos recentes e monitoramento de ordens de compra/venda.
- **Estética Premium**: Interface inspirada em Awwwards, utilizando Dark Mode, Glassmorphism e layouts em Bento Grid.

## 🛠️ Tecnologias Utilizadas

- **Front-end**: React.js
- **Estilização**: Tailwind CSS / Lucide React (Ícones)
- **Gráficos**: Recharts
- **Roteamento**: React Router DOM
- **Back-end**: Node.js & Express (API Rest)
- **Banco de Dados**: MongoDB
